# Printer Troubleshooter Pro

**Complete printer fix with automatic Print Spooler restart** - The ultimate solution for stuck print jobs, offline printers, and connection issues.

## 🚀 What It Fixes

| Problem | Success Rate |
|---------|--------------|
| Print jobs stuck in queue | 95% |
| Printer offline/not responding | 90% |
| Network printer connection | 85% |
| Driver issues | 80% |

## ⚡ Key Features

- ✅ Auto Print Spooler restart (main fix)
- ✅ Clear stuck print jobs automatically
- ✅ Check network printer ports (9100, 515, 631, 80)
- ✅ Verify SNMP status
- ✅ Reset printer driver/port
- ✅ Full 7-step diagnostic

## 📋 Requirements

- Windows 10/11
- PowerShell 5.1+
- **Administrator privileges** (required for service restart)

## 🛠 How to Use

```powershell
# Open PowerShell as Administrator
.\Printer-Troubleshooter-Pro.ps1
```
